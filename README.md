# Презентация: Криптографические алгоритмы

## Описание проекта
Интерактивная презентация по криптографическим алгоритмам, выполненная с использованием фреймворка Reveal.js. Презентация включает теоретический материал, примеры и работающие демонстрации алгоритмов.

## Требования к проекту
- [x] Презентация выполнена с использованием Markdown, где это возможно
- [x] Использован фреймворк Reveal.js
- [x] Тема оформления вынесена в отдельный CSS файл
- [x] Использован JavaScript для реализации сценариев
- [x] Минимальное количество слайдов: 20
- [x] Перед каждым разделом используется слайд-разделитель
- [x] Оформление презентации читаемо
- [x] Имеются титульный и заключительный слайды
- [x] Реализовано меню для быстрого перехода к слайду
- [x] Реализована навигация: стрелки, номер слайда
- [x] Наличие README с заданием

## Реализованные алгоритмы

### 1. Шифр Цезаря
- **Описание**: Алгоритм подстановки с фиксированным сдвигом
- **Алфавит**: 47 символов (33 русские буквы + 10 цифр + 4 знака препинания)
- **Реализация**: Полностью рабочая демонстрация с возможностью шифрования/дешифрования

### 2. Криптосистема Меркла-Хеллмана
- **Описание**: Асимметричная криптосистема на основе задачи о рюкзаке
- **Реализация**: Генерация ключей, шифрование и дешифрование бинарных сообщений

### 3. Азбука Морзе
- **Описание**: Способ кодирования символов с помощью точек и тире
- **Алфавит**: Полный русский алфавит, цифры и знаки препинания
- **Реализация**: Двусторонний конвертер текст↔Морзе
