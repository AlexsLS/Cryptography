Часть 1

Введение в криптографию

Что такое криптография?

Криптография (от греч. κρυπτός — скрытый и γράφω — пишу) — наука о методах обеспечения конфиденциальности, целостности данных, аутентификации и шифрования.

Современная криптография включает симметричные и асимметричные алгоритмы, хеш-функции, цифровые подписи и протоколы обмена ключами.

Основные цели криптографии

Конфиденциальность

Защита информации от несанкционированного доступа. Только авторизованные стороны могут читать данные.

Целостность

Обеспечение неизменности информации при передаче и хранении. Обнаружение любых изменений данных.

Аутентификация

Подтверждение подлинности отправителя и получателя. Установление доверия между сторонами.

Неотрекаемость

Невозможность отказа от авторства. Гарантия, что отправитель не сможет отрицать отправку сообщения.

Часть 2

Шифр Цезаря

Историческая справка

Один из древнейших шифров, названный в честь римского императора Гая Юлия Цезаря (100-44 до н.э.).

Использовался для секретной переписки с генералами во время военных кампаний. Цезарь применял сдвиг на 3 позиции.

Описание шифра Цезаря

Принцип работы:

  • Алгоритм подстановки с фиксированным сдвигом
  • Каждая буква сдвигается на N позиций в алфавите
  • Пример при сдвиге 3: A→D, B→E, C→F
  • Математическая формула: E(x) = (x + k) mod n
  • Дешифровка: D(x) = (x - k) mod n

Алфавит в нашей реализации: 47 символов (33 русские буквы + 10 цифр + пробел, запятая, точка, знак вопроса, восклицательный знак)

Часть 3

Задача о рюкзаке

Описание задачи:

Задача о рюкзаке — комбинаторная задача оптимизации: имеется рюкзак определённой грузоподъёмности и набор предметов с весами и ценностями.

Требуется выбрать подмножество предметов, чтобы их суммарный вес не превышал грузоподъёмность, а суммарная ценность была максимальной.

Криптосистема Меркла-Хеллмана

История и особенности:

  • Разработана в 1978 году Ральфом Мерклом и Мартином Хеллманом
  • Одна из первых асимметричных криптосистем
  • Основана на NP-полной задаче о рюкзаке
  • Использует супервозрастающие последовательности
  • Была взломана в 1984 году Ади Шамиром

Примечание: Хотя система была взломана, она представляет исторический интерес как первая попытка создать асимметричную криптосистему на основе задачи о рюкзаке.

Демонстрация: Меркл-Хеллман

Введите бинарную строку (только 0 и 1) для шифрования. Длина сообщения определяет размер ключа.

Азбука Морзе

Историческая справка:

  • Разработана Сэмюэлем Морзе в 1838 году совместно с Альфредом Вейлом
  • Первое сообщение: "What hath God wrought" (1844)
  • Использовалась для телеграфной связи на большие расстояния
  • Актуальна в радиосвязи, на флоте и в авиации
  • Международный сигнал бедствия: SOS (··· --- ···)

Реализация алгоритмов на JavaScript

Шифр Цезаря:

const CAESAR_ALPHABET = 'АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ 1234567890,.?!';

function caesarCipher(text, shift, encrypt = true) {
    let result = '';
    const n = CAESAR_ALPHABET.length;
    
    for (let char of text.toUpperCase()) {
        const index = CAESAR_ALPHABET.indexOf(char);
        if (index === -1) {
            result += char;
            continue;
        }
        
        let newIndex;
        if (encrypt) {
            newIndex = (index + shift) % n;
        } else {
            newIndex = (index - shift + n) % n;
        }
        
        result += CAESAR_ALPHABET[newIndex];
    }
    
    return result;
}
