# Презентация по криптографическим алгоритмам

## Задание

Создать презентацию с использованием Markdown, где это возможно, на тему криптографических алгоритмов.

## Требования

1. Презентация должна быть выполнена с помощью Markdown, где это возможно
2. Можно выполнить на любом понравившемся фреймворке или написать собственный
3. Тема(оформление) обязательно должна быть вынесена в отдельный CSS файл
4. Для написания сценария разрешается использовать любой удобный для Вас фреймворк
5. Минимальное количество слайдов: 20
6. Перед каждым разделом необходимо использовать слайд-разделитель
7. Оформление презентации должно быть читаемым
8. Обязательными являются титульный слайд, заключительный слайд
9. Обязательно наличие меню для быстрого перехода к слайду
10. Обязательно наличие навигации: стрелки, номер слайда
11. Обязательно наличие readme в корне презентации с заданием

## Обязательные слайды

- Определение криптографии
- Описание шифра Цезаря
- Пример шифра Цезаря
- Программа, написанная на языке JavaScript, реализующий шифр Цезаря (Использовать буквы русского алфавита, цифры, пробел, запятая, точка, знак вопроса, всего 47 символов)
- Задача о рюкзаке. Описание.
- Криптосистема Меркла-Хеллмана. Описание и пример
- Программа, написанная на языке JavaScript, реализующую алгоритм Меркла-Хеллмана
- Азбука Морзе. Описание и алфавит
- Программа, написанная на языке JavaScript по конвертации текста, написанного на русском языке на азбуку Морзе и наоборот
- Исходный код программ

## Реализация

Презентация реализована с использованием фреймворка Reveal.js, который поддерживает Markdown и предоставляет возможности для навигации и красивого оформления.

Все алгоритмы реализованы на JavaScript и интегрированы в презентацию.
