Введение в криптографию

Что такое криптография?

Криптография (от греч. κρυπτός — скрытый и γράφω — пишу) — наука о методах обеспечения конфиденциальности, целостности данных, аутентификации и шифрования.

Основные цели:

  • Конфиденциальность - защита от несанкционированного доступа
  • Целостность - обеспечение неизменности информации
  • Аутентификация - подтверждение подлинности
  • Неотрекаемость - невозможность отказа от авторства

Историческая справка

Древний мир

Шифр Цезаря (100-44 до н.э.)

Средневековье

Моноалфавитные шифры

XX век

Энигма, DES, RSA

Современность

AES, квантовая криптография

Ключевые моменты:

  • Эволюция от простых подстановок до сложных математических алгоритмов
  • Появление асимметричной криптографии в 1970-х
  • Современные стандарты: AES (2001), RSA (1977)

Часть 2

Шифр Цезаря

Шифр Цезаря: Описание

Шифр Цезаря — один из древнейших шифров, названный в честь римского императора Гая Юлия Цезаря.

Принцип работы:

  • Алгоритм подстановки, где каждая буква сдвигается на фиксированное число позиций в алфавите
  • Простой пример: сдвиг на 3 позиции: A → D, B → E, C → F, ...
  • Математическая формула: E(x) = (x + k) mod n
  • Дешифровка: D(x) = (x - k) mod n

Примечание: В нашей реализации используется алфавит из 47 символов

Шифр Цезаря: Пример

Исходный текст:

ПРИВЕТ, МИР!

Сдвиг на 3 позиции:

ТУЛЗИХ, ПЛУ!

Визуализация сдвига:

П → Т Р → У И → Л В → З Е → И Т → Х

Алфавит: 33 русские буквы + 10 цифр + пробел, запятая, точка, знак вопроса = 47 символов

Часть 3

Задача о рюкзаке и Меркл-Хеллман

Задача о рюкзаке

Задача о рюкзаке — комбинаторная задача: имеется рюкзак определённой грузоподъёмности и набор предметов с определёнными весами и ценностями. Требуется определить, какие предметы взять, чтобы максимизировать общую ценность, не превышая грузоподъёмность.

В криптографии:

  • Используется для создания односторонних функций
  • Основа для криптосистемы Меркла-Хеллмана
  • Простая в одну сторону, сложная в обратную (NP-полная задача)

Пример:

Рюкзак вмещает 15 кг. Предметы: (5кг, $10), (7кг, $13), (6кг, $8), (3кг, $6)

Криптосистема Меркла-Хеллмана

Криптосистема Меркла-Хеллмана — одна из первых асимметричных криптосистем, основанная на задаче о рюкзаке (1978).

Принцип работы:

  1. Выбор сверхрастущей последовательности
  2. Выбор множителя и модуля для преобразования
  3. Публичный ключ: преобразованная последовательность
  4. Приватный ключ: исходная последовательность, множитель и модуль

Важно: Система была взломана в 1984 году, но представляет исторический интерес

Пример: Меркл-Хеллман

Шаг 1: Сверхрастущая последовательность

[2, 5, 11, 23, 47, 95, 203]

Шаг 2: Выбираем множитель и модуль

Множитель: m = 41, Модуль: n = 491

Шаг 3: Вычисляем публичный ключ

[82, 205, 451, 443, 418, 359, 372]

Шаг 4: Шифрование сообщения

Сообщение "1010110" → Сумма: 82 + 451 + 418 + 359 = 1310

Демонстрация: Меркл-Хеллман

Азбука Морзе

Азбука Морзе: Описание

Азбука Морзе — способ кодирования букв алфавита, цифр и знаков препинания с помощью коротких (точка) и длинных (тире) сигналов.

Историческая справка:

  • Разработана Сэмюэлем Морзе в 1838 году
  • Использовалась для телеграфной связи
  • Актуальна в радиосвязи, авиации, на флоте
  • Международный сигнал бедствия: SOS (··· --- ···)

Исходный код: Шифр Цезаря


// Алфавит из 47 символов
const CAESAR_ALPHABET = 'АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ 1234567890,?.';

function caesarCipher(text, shift, encrypt = true) {
    let result = '';
    const n = CAESAR_ALPHABET.length;
    
    for (let char of text.toUpperCase()) {
        const index = CAESAR_ALPHABET.indexOf(char);
        if (index === -1) {
            result += char;
            continue;
        }
        
        let newIndex;
        if (encrypt) {
            newIndex = (index + shift) % n;
        } else {
            newIndex = (index - shift + n) % n;
        }
        
        result += CAESAR_ALPHABET[newIndex];
    }
    
    return result;
}

// Пример использования
const encrypted = caesarCipher('ПРИВЕТ', 3, true);
const decrypted = caesarCipher(encrypted, 3, false);

Исходный код: Меркл-Хеллман


class MerkleHellman {
    constructor() {
        this.privateKey = this.generateSuperIncreasingSequence(8);
        this.multiplier = this.generateCoprime(100);
        this.modulus = this.generateModulus(this.privateKey, this.multiplier);
        this.publicKey = this.generatePublicKey();
    }
    
    generateSuperIncreasingSequence(n) {
        const sequence = [1];
        for (let i = 1; i < n; i++) {
            sequence.push(sequence.reduce((a, b) => a + b, 0) + 1);
        }
        return sequence;
    }
    
    generatePublicKey() {
        return this.privateKey.map(num => 
            (num * this.multiplier) % this.modulus
        );
    }
    
    encrypt(message, publicKey) {
        let sum = 0;
        for (let i = 0; i < message.length; i++) {
            if (message[i] === '1') {
                sum += publicKey[i];
            }
        }
        return sum;
    }
    
    decrypt(ciphertext) {
        // Реализация дешифрования с использованием приватного ключа
        // и расширенного алгоритма Евклида
        return this.solveKnapsack(ciphertext, this.privateKey);
    }
}

const MORSE_CODE = {
    'А': '.-', 'Б': '-...', 'В': '.--', 'Г': '--.', 'Д': '-..',
    'Е': '.', 'Ё': '.', 'Ж': '...-', 'З': '--..', 'И': '..',
    'Й': '.---', 'К': '-.-', 'Л': '.-..', 'М': '--', 'Н': '-.',
    'О': '---', 'П': '.--.', 'Р': '.-.', 'С': '...', 'Т': '-',
    'У': '..-', 'Ф': '..-.', 'Х': '....', 'Ц': '-.-.', 'Ч': '---.',
    'Ш': '----', 'Щ': '--.-', 'Ъ': '--.--', 'Ы': '-.--', 'Ь': '-..-',
    'Э': '..-..', 'Ю': '..--', 'Я': '.-.-',
    '1': '.----', '2': '..---', '3': '...--', '4': '....-',
    '5': '.....', '6': '-....', '7': '--...', '8': '---..',
    '9': '----.', '0': '-----',
    ' ': '/', ',': '--..--', '.': '.-.-.-', '?': '..--..'
};

function textToMorse(text) {
    return text.toUpperCase().split('').map(char => 
        MORSE_CODE[char] || char
    ).join(' ');
}

function morseToText(morse) {
    const morseToChar = {};
    for (const [char, code] of Object.entries(MORSE_CODE)) {
        morseToChar[code] = char;
    }
    
    return morse.split(' ').map(code => 
        morseToChar[code] || code
    ).join('');
}
