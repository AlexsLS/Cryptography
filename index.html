Часть 1

Введение в криптографию

Что такое криптография?

Криптография (от греч. κρυπτός — скрытый и γράφω — пишу) — наука о методах обеспечения конфиденциальности, целостности данных, аутентификации и шифрования.

Основные цели криптографии

Конфиденциальность

Защита информации от несанкционированного доступа

Целостность

Обеспечение неизменности информации при передаче

Аутентификация

Подтверждение подлинности отправителя

Неотрекаемость

Невозможность отказа от авторства

Шифр Цезаря

Историческая справка

Один из древнейших шифров, названный в честь римского императора Гая Юлия Цезаря.

Использовался для секретной переписки с генералами.

Описание шифра Цезаря

Принцип работы:

  • Алгоритм подстановки с фиксированным сдвигом
  • Каждая буква сдвигается на N позиций в алфавите
  • Пример при сдвиге 3: A→D, B→E, C→F
  • Математическая формула: E(x) = (x + k) mod n
  • Дешифровка: D(x) = (x - k) mod n

Алфавит в нашей реализации: 47 символов (33 русские буквы + 10 цифр + 4 знака препинания)

Демонстрация: Шифр Цезаря

Алфавит включает: русские буквы, цифры, пробел, запятую, точку, знак вопроса

Часть 3

Задача о рюкзаке

Описание задачи:

Комбинаторная задача оптимизации: имеется рюкзак определённой грузоподъёмности и набор предметов с весами и ценностями. Требуется максимизировать ценность, не превышая грузоподъёмность.

Криптосистема Меркла-Хеллмана

История:

  • Разработана в 1978 году Ральфом Мерклом и Мартином Хеллманом
  • Одна из первых асимметричных криптосистем
  • Основана на задаче о рюкзаке
  • Была взломана в 1984 году, но представляет исторический интерес

Демонстрация: Меркл-Хеллман

Введите бинарную строку (только 0 и 1) для шифрования

Реализация алгоритмов на JavaScript

Шифр Цезаря:

const CAESAR_ALPHABET = 'АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ 1234567890,?.';

function caesarCipher(text, shift, encrypt = true) {
    let result = '';
    const n = CAESAR_ALPHABET.length;
    
    for (let char of text.toUpperCase()) {
        const index = CAESAR_ALPHABET.indexOf(char);
        if (index === -1) {
            result += char;
            continue;
        }
        
        let newIndex;
        if (encrypt) {
            newIndex = (index + shift) % n;
        } else {
            newIndex = (index - shift + n) % n;
        }
        
        result += CAESAR_ALPHABET[newIndex];
    }
    
    return result;
}
